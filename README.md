## Project Overview
This system provides a streamlined API for managing organizational resources, focusing on the "Service Layer" pattern to ensure business logic remains decoupled and highly testable.

### Key Functionalities:
* RESTful Resource Management: Full CRUD operations for staff and organizational data.
* Data Persistence: Integrated with Spring Data JPA for seamless database interactions.
* Request Validation: Robust handling of incoming data to ensure system integrity.

---

## Tech Stack
* Language: Java
* Framework: Spring Boot (Web, Data JPA)
* Build Tool: Maven
* Database: H2 / PostgreSQL (Configurable)
* Architecture: Layered Pattern (Controller, Service, Repository, Model)
